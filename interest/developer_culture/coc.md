## Code of Conduct
https://en.wikipedia.org/wiki/Code_of_conduct

한국어로는 행동 지침, 행동 강령 등으로 번역하는 code of conduct는 말 그대로 '어떻게 행동하라'는 권고이다.
이제껏 만나본 CoC들은 어떤 행사나 단체를 운영할 때, 그 안에서 상호 존중과 원활한 상호작용이 일어날 수 있도록 노력하는 내용이었다.
